翻着过往收集的笔记时，突然发现了这些问题，刚好适用于每次思考新机会，做个记录...

1.产品要解决什么问题?(产品价值)

2.为谁解决这个问题?(目标市场)

3.成功的机会有多大?(市场规模)

4.怎样判断产品成功与否?(度量指标或收益指标)

5.有哪些同类产品?(竞争格局)

6.为什么我们适合做这个产品?(竞争优势)

7.时机合适吗?(市场时机)

8.如何把产品推向市场?(营销组合策略)

9.成功的必要条件是什么?(解决方案要满足的条件)

10.根据以上问题，给出评估结论(继续或放弃)
